# DelphiProfileInit

Utilitário para modificar initialization de todas as units de 
programa delphi para medição de desempenho.

Esse utilitário percorre todas as units de um diretório delphi
e cria (ou modifica) a seção initialization para usar a 
unit unPerfCounter para medição do desempenho.

Veja o artigo completo sobre análise de desempenho de inicialização
no meu site: http://matheusd.com/post/Profile_Delphi_Startup/
